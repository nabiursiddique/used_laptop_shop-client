# used_laptop_shop


## Tech Stack

**Client:** React

**Server:** Node, Express

**Database:** MongoDB


## Technology Used

- React
- React Router DOM
- React Hook Form
- React Query/Tanstack Query 
- React Stripe JS 
- React Icon
- React Hot Toast
- Tailwind CSS
- Tailwind CSS Animatied
- Daisy UI
- Date fns
- Firebase Authentication
- Dotenv
- Image hosting api


## Features
- Creating/Sign up a user
- Sign In a user and Signout a user
- Admin can make other users admin 
- Admin can delete other user
- Seller can add their product 
- Buyer can book a product
- Responsive website
